### CI-CD Pipeline with Kubernetes

This walks you through settings up a simple CI-CD pipeline with [Docker](https://docs.docker.com/install/), [Kubernetes](https://kubernetes.io/docs/setup/), and [Jenkins](https://jenkins.io/doc/pipeline/tour/getting-started/) using  [Minikube](https://kubernetes.io/docs/tasks/tools/install-minikube/).


#### Prerequisites
- Install [Virtualbox](https://www.virtualbox.org/wiki/Downloads)
- Install [Minikube](https://kubernetes.io/docs/tasks/tools/install-minikube/)
- Install [kubectl](https://kubernetes.io/docs/tasks/tools/install-kubectl/)
- Install [Docker](https://docs.docker.com/install/)
